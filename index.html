Fix Windows Update issues

A step-by-step guide to resolving common problems that prevent Windows from downloading or installing updates.

Windows 11 · Windows 10 · Last updated: October 2024
Note If you're seeing a specific error code, you can search for it in the search box above. Common codes include 0x80070002, 0x800f0922, 0x80240438, and 0x80070422.

Before you begin

Windows Update keeps your device secure and running smoothly by downloading and installing the latest security patches, feature updates, and driver updates. If updates fail to install or get stuck during download, use the steps in this article to get things working again.

Before starting any troubleshooting, make sure of the following:

  • Your device is connected to the internet and the connection is stable.
  • You're signed in with an administrator account.
  • Your device has at least 10 GB of free disk space.
  • Your device's date and time are set correctly.
  • Any third-party antivirus software is temporarily disabled (some can interfere with updates).

Step 1: Run the Windows Update Troubleshooter

The built-in troubleshooter can automatically detect and fix many common update problems. Try this first—it resolves the majority of issues without any further action.

1
Open Settings by pressing Windows key + I.
2
Go to System > Troubleshoot > Other troubleshooters.
3
Find Windows Update and select Run.
4
Follow the on-screen prompts and apply any recommended fixes.
5
Restart your device and try checking for updates again.
Tip
The troubleshooter may ask you to restart your device before it can apply certain fixes. Save your work before continuing.

Step 2: Restart Windows Update services

Windows Update relies on several background services. If any of these services stop running or become stuck, updates can fail. You can restart them manually.

1
Press Windows key + R, type services.msc, and press Enter.
2
Locate the following services:
  • Background Intelligent Transfer Service (BITS)
  • Windows Update
  • Cryptographic Services
  • Windows Installer
3
Right-click each service, select Restart (or Start if it's not running).
4
Close the Services window and try updating again.

Step 3: Reset Windows Update components

If the troubleshooter and service restart don't resolve the issue, resetting the update components can help. This clears the update cache and re-registers the update DLLs.

Open Command Prompt as administrator, then run the following commands one at a time, pressing Enter after each:

net stop wuauserv net stop cryptSvc net stop bits net stop msiserver ren C:\Windows\SoftwareDistribution SoftwareDistribution.old ren C:\Windows\System32\catroot2 catroot2.old net start wuauserv net start cryptSvc net start bits net start msiserver

After running these commands, restart your device and check for updates again.

Warning Don't delete the SoftwareDistribution.old or catroot2.old folders manually. Windows will recreate the necessary folders automatically. If everything works, you can delete the .old folders after a few days.

Step 4: Manually install updates

If Windows Update still isn't working, you can download and install updates manually from the Microsoft Update Catalog.

  1. Find the Knowledge Base (KB) number of the update that's failing. You'll see it in the update error message or in Settings > Windows Update > Update history.
  2. Go to catalog.update.microsoft.com in your browser.
  3. Search for the KB number.
  4. Download the version that matches your device architecture (x64 for most modern PCs, ARM64 for ARM devices).
  5. Run the downloaded file to install the update.
  6. Restart your device when prompted.

If you've tried all the steps above and Windows Update still isn't working, you can get help directly from Microsoft. Our support agents are available to assist you with complex update issues.
